# Any% 

With well over 1,000 runs on Speedrun.com, this is by far the most popular and competitive category in NSMBW. It is also the most beginner-friendly, making it a great place to start learning this speedrun.

Before hopping into a run, we recommend taking a look at each tutorial and getting confident with (at least) beginner strats in every level.

## Category Specific Information

Beat the game as fast as possible. Read the game and category rules on [Speedrun.com](speedrun.com/nsmbw) for more information.

This category retimes runs **under 25 minutes** (done on physical console).
