# Documents

This page contains all **up-to-date** NSMBW speedrunning documents. These are all open for the public to view and creators are credited if you want to reach out for more information.

---

[Community IL Tracker & Leaderboard](https://docs.google.com/spreadsheets/d/1afEAae4S_rL3FzVB9HbkJck_w3F-MIjn2F-05QCKxX8/edit?usp=sharing){: .btn }

This spreadsheet includes an official current tally of current Individual Level world record count, as well as individual speedrunners' progress. If you are interested in being added hit up one of the creators!

**Creator(s):** Cads, BigCheese, Volza

---

[Unofficial Emulator Leaderboard](){: .btn }

The official/unofficial leaderboards for runs done on emulator. Has a separate sheet for each main category and category extension.

**Creator(s):** LunaEclipse, Toogs

---

[Precise Any% IGT Leaderboard](https://docs.google.com/spreadsheets/d/1ssq1OBuJ6XMzaCDjRFNZfBFYbQjxXYHlXUqvR-ZUpS0/edit?usp=sharing){: .btn }

This sheet tracks precise IGT records for Any% stages and ranks runners by sum and individual level efforts.

**Creator(s)** Azord
