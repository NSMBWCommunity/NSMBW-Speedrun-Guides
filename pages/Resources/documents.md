# Documents

This page contains all **up-to-date** NSMBW speedrunning documents. These are all open for the public to view and creators are credited if you are wanting to reach out for more information.

---

[Official IL Leaderboards](){: .btn }

This spreadsheet includes an official current tally of current Individual Level world record count, as well as individual speedrunners' progress. If you are interested in being added hit up one of the creators!

**Creator(s):** Cads, BigCheese, Volza

---

[Unofficial Emulator Leaderboard](){: .btn }

The official/unofficial leaderboards for runs done on emulator. Has a separate sheet for each main category and category extension.

**Creator(s):** LunaEclipse, Toogs

---

[NSMBW WRs](){: .btn }

The sheet used to track main board world records (with notes), world records per speedrunner, and sub-25 Any% times per country.

**Creator(s):** BMAN, LunaEclipse, Kaoskarl, gasping-muffin

---

[Any% IGT Leaderboard](){: .btn }

This sheet tracks precise IGT records for Any% stages only and ranks runners by sum and individual level efforts.

**Creator(s)** Azord

---

[Any% Top 30 IGTs](){: .btn }

A sheet that compares IGTs in each Any% stage across the top 30 on SRC. Ranked by sum of IGTs.

**Creator(s):** Azord
