# Discord Servers

Here is a list of Discord Servers that may aid you in your NSMBW speedrunning journey.

We recommend at minimum joining the main speedrunning server!

---

[NSMBW Speedrunning](https://discord.gg/pMKAaGscrh){: .btn }

The main hub for sharing strats, PBs, and discussing speedrunning this game in general.

---

[NSMBW Tutorials](https://discord.gg/xMmfPyJ6Mh){: .btn }

This server is used for website documentation, discussion, and resource development.

---

[NSMBW Tournaments Server](https://discord.gg/uWnjt8FCVA){: .btn }

All NSMBW tournament information can be found here! If you're interested in signing up or learning when the next tournament is, join and ask a staff member!

---

[NSMBW BTT Server](https://discord.gg/uhyVmA5DuE){: .btn }

The community BTT server for posting clips to create a "best theoretical time" segmented speedrun.

---

[NSMB Series Speedrunning](https://discord.gg/GZ5Gg5E7RJ){: .btn }

Interested in speedrunning another game in the NSMB Series like New Super Mario Bros. DS? Join here!

---

## Inactive Servers

Check out the [NSMBW Speedrunning Archive's Inactive Servers](https://nsmbwcommunity.github.io/NSMBW-Speedrun-Guides/resources/nsmbw-speedrunning-archive/) for inactive servers you may be interested in.
