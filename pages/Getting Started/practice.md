# Practice

Knowing how to effectively practice NSMBW is what will keep you from plateauing and help you to continuously improve your times.

## Practice Mod
The practice mod for NSMBW can be used on all platforms with homebrew set up.
Homebrew is really easy to set up on [Wii](https://youtu.be/bHGf7JbQR9c) and is fairly simple on [WiiU](https://www.youtube.com/watch?v=iTJ3cShA6AI).

All NSMBW mods use a Homebrew app called 
[Riivolution](https://aerialx.github.io/rvlution.net/riivolution.zip).
To use this mod download and open the latest version of the practice mod zip/rar located in the [Server Resources Channel](https://discordapp.com/channels/135832360099774464/1033932077890027561)

### Installation
Drag both folders onto the root of your SD card and hit replace files if a prompt shows up.

## Info Display
MGR's Info Display is a very useful tool to have while practicing.

This display uses the empty space of the 4:3 resolution to dispaly important info about mario's current state.

**Features:**
- Precice IGT
- Input Display
- Mario's X and Y Speed
- Mario's Current state (jumping, falling, sliding, etc)
- Slide Timer / Propeller Timer

### Installation
The newest version is pinned in the general channel of the NSMBW Server.
Drag both folders onto the Root of your SD card and hit replace files if a prompt shows up.
