## Practice Mod
The Practice Mod is the **best** way to practice most aspects of this game.

NSMBW mods require installing [Homebrew](https://wiibrew.org/wiki/Homebrew_setup) to your Wii/Wii U.

Linked are easy-to-follow Homebrew Setup Tutorials for: 
- [Wii](https://youtu.be/bHGf7JbQR9c)
- [Wii U](https://www.youtube.com/watch?v=iTJ3cShA6AI).

The Practice Mod also works well with [Dolphin Emulator](https://dolphin-emu.org/). ([Tutorial](https://www.youtube.com/watch?v=Jt97J7aWf8k) on how to play Riivolution patches on Dolphin)

All NSMBW mods launch using a Homebrew app called 
[Riivolution](https://aerialx.github.io/rvlution.net/wiki/Riivolution/).

## Info Display
MGR's Info Display is a very useful tool to have while practicing.

This display uses the empty space of the 4:3 resolution to dispaly important info about mario's current state.

**Features:**
- Precice IGT
- Input Display
- Mario's X and Y Speed
- Mario's Current state (jumping, falling, sliding, etc)
- Slide Timer / Propeller Timer

### Installation
The newest version is pinned in the general channel of the NSMBW Server.
Drag both folders onto the Root of your SD card and hit replace files if a prompt shows up.
